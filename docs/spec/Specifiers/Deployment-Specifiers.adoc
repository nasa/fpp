=== Deployment Specifiers

A *deployment specifier* specifies a deployment as part of a
<<Definitions_System-Definitions,system definition>>.
A deployment is an executable program described by a
<<Definitions_Topology-Definitions,topology definition>>.

==== Syntax

`deployment`
<<Scoping-of-Names_Qualified-Identifiers,_qual-ident_>>

==== Semantics

The qualified identifier must
<<Scoping-of-Names_Resolution-of-Qualified-Identifiers,refer to>>
a
<<Definitions_Topology-Definitions,topology definition>>.

==== Examples

[source,fpp]
----
deployment T
deployment A.B
----
