=== Sizeof Expressions

A *sizeof expression* represents the size of the storage associated with a type.

==== Syntax

`sizeof` `(` <<Type-Names,_type-name_>> `)`

==== Semantics

* The type name must <<Scoping-of-Names_Resolution-of-Qualified-Identifiers,refer to>>
a type stem:[T].

* The type stem:[T] must be <<Types_Displayable-Types,displayable>>.

* The expression evaluates to the <<Types_Serialized-Sizes, serialized size>> of _T_.

==== Example

[source,fpp]
----
type T = U32
constant a = sizeof(T) # sizeof(T) evaluates to 4

constant b = sizeof(F64) # sizeof(F64) evaluates to 8

array A = [3] U16
constant c = sizeof(A) # sizeof(A) evaluates to 6
----
