=== String Concatenation Expressions

A *string concatenation expression* concatenates two string values.

==== Syntax

_s_~1~ `+` _s_~2~

==== Semantics

1. Both _s_~1~ and _s_~2~ must have
<<Type-Names_String-Type-Names, string type>>.

2. The value of the expression is the string formed by appending
the characters of _s_~2~ to the characters of _s_~1~.

==== Example

[source,fpp]
----
constant c = "a" + "b" + "c"    # value is "abc"
constant a = "hello " + c + "!" # value is "hello abc!"
----
