=== System Definitions

A *system definition* defines an embedded software system.
When a system definition appears in the model, it
causes code and a ground dictionary to be generated.

==== Syntax

`system`
<<Lexical-Elements_Identifiers,_identifier_>>
`:`
<<Scoping-of-Names_Qualified-Identifiers,_qual-ident_>>

==== Semantics

. An FPP model may contain at most one system definition.

. The qualified identifier must
<<Scoping-of-Names_Resolution-of-Qualified-Identifiers,refer to>>
a
<<Definitions_Topology-Definitions,topology definition>>.

==== Examples

[source,fpp]
----
@ The spacecraft system
system Spacecraft: SpacecraftTopology

@ The topology for the system
topology SpacecraftTopology { ... }
----
